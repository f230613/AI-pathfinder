# Quickstart Guide

AI Pathfinder --- Uninformed Search Visualizer

## 1. Install Python

Make sure Python 3.8 or higher is installed.

Check version:

python --version

## 2. Install Required Library

Install matplotlib:

pip install matplotlib

Or if using requirements file:

pip install -r requirements.txt

## 3. Run the Program

Open terminal inside the project folder and run:

python ai_pathfinder.py

## 4. Choose Algorithm

After running the script, a menu will appear:

1.  Breadth-First Search
2.  Depth-First Search
3.  Uniform Cost Search
4.  Depth-Limited Search
5.  Iterative Deepening DFS
6.  Bidirectional Search

Enter a number between 1--6 to run an algorithm.

## 5. View Results

-   The final path will be printed in the console.
-   The animated grid window will open automatically.
-   Statistics such as nodes explored and path length will be shown.

## Troubleshooting

If matplotlib is not installed, run:

pip install matplotlib

If python command does not work, try:

python3 ai_pathfinder.py
